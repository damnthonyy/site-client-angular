# Client Site Angular avec intégration Osmow

Application Angular minimale avec intégration Osmow.

## Installation

```bash
npm install
```

## Démarrage

```bash
npm start
```

L'application sera accessible sur `http://localhost:3004`

## Structure

- `/blog` - Page blog avec intégration Osmow
- `/blog/:slug` - Page article avec intégration Osmow
